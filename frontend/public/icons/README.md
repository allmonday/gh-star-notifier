# Icons Setup

This directory should contain PWA icon files.

## Quick Solution: Use Online Generator

1. Go to https://realfavicongenerator.net/
2. Upload your logo (any image)
3. Download the generated icons
4. Extract and copy the icon files to this directory

## Required Icon Files

- favicon.ico
- favicon-16x16.png
- favicon-32x32.png
- favicon-96x96.png
- favicon-128x128.png
- icon-128x128.png
- icon-192x192.png
- icon-256x256.png
- icon-384x384.png
- icon-512x512.png

## For Testing

For now, you can ignore the 404 errors for missing icons.
The PWA will still work without custom icons.
